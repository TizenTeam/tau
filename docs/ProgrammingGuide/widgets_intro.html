Introduction for using widgets

What is widget?

Widgets are predefined components consisting web application for developers to use. Widgets have their own shapes, themes, and behaviors.

Putting widgets in your html code

To use widgets in your app, simply add widget element code into your html code. For example, if you want to put a listview into content area, just add following code, 


<ul data-role="listview">
	<li>listitem 1</li>
	<li>listitem 2</li>
</ul>
and the output will contain listview widget, like below.

  • listitem 1
  • listitem 2

Putting widgets in your javascript code

Dynamically putting widgets in your javascript code is somewhat different from putting them in html code. Widget tags in html code are parsed when current page is created('pagecreate' event). This means that all tags which do not exist at page creating time will not be parsed to widget elements by default.

For example, when you try to put a listview by following code, 


var container = $('#listview_container');
container.append($('<ul data-role="listview"><li>listitem 1</li></ul>'));
