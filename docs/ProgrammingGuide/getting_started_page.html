Page structure

Page

A page is a div block having data-role="page", such like <div data-role="page">.

Page structure

A page has sub elements; header, content, and footer. Header and footer can be omitted.

header

content

footer

header

A header is a div block, having data-role="header".
Header places on top of the page, showing page title, back button, etc.
This header can have several option attributes below;

  • data-add-back-btn="true" : Show back button on header. Default option is false.

content

A content is a div block, having data-role="content". This is an area below header, which has all contents in the page. Any objects, any widgets can be in it.
