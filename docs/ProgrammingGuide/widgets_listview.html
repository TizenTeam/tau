Listview

Overview

A listview is an ul element having attribute data-role="listview", such as <ul data-role="listview">.
With the attribure, an ul element becomes a listview, which has its own look&feel and behaviors.

listview styles

Listview items, which are actually <li> elements, can have many kinds of styles. As Tizen web UI framework provides some predefined listview item styles for app developer's convenience, you can and use those styles if you find a good one you wanted to make. Or, you can create your own style by writing a class for <li> items.
