Inside template code

A sample application generated by template generator is like below. 

<html>
  <head>
    <title>My App</title>
    <!--NOTE:
      jquery.js and tizen-web-ui-fw.js must be included.
      DO NOT REMOVE below code!
    -->
    <script src="tizen-web-ui-fw/jquery.js"></script>
    <script src="tizen-web-ui-fw/tizen-web-ui-fw.js"
      data-framework-root="tizen-web-ui-fw"
      data-framework-version="0.1" 
      data-framework-theme="nbeat-black"></script>

    <!--NOTE:
      Additional scripts and css files are to be placed here.
      You can use jQuery namespace($) and all functionalities in jQuery
      in your script.  For example:

        <script src="main.js"></script>
        <link rel="stylesheet" href="my.css">

      When you want to manipulate elements in your code, you have to
      use $(document).ready() for your code to work properly, like this;

        $(document).ready(function(){ ... });
    -->
    </head>

    <body>
        <div data-role="page" id="template-page">
            <div data-role="header">
                <h1>Web UI Framework - Template application</h1>
            </div>
            <div data-role="content">
                <p>This application is an empty starting point.</p>
            </div> <!-- /content -->
        </div> <!-- /page -->
    </body>
</html>

HTML Header

The <title> tag contains the title of this application.

The next two <script> tags load basic Tizen web UI freamework libraries. First one is jQuery, and the other one contains jQuery mobile, other small libraries, Tizen widgets, and a loader. Those two javascripts must be included for a Tizen app.

For a <script> tag including tizen-web-ui-fw.js, you can add some options describing framework's root directory, version, and default theme, as you can see in this example.

You can put additional <script src="..."> or <link rel="stylesheet" src="..."> tags for your own scripts or stylesheets. It is recommended to put them after default two <script> tags, as you can use any jQuery, jQueryMobile, and Tizen widget APIs provided from these two.

In <body> section, There are one or more pages, which contain header, content, and footer. A guide explaining the structure of a page is in next section.
