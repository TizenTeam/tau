Running your javascript code

To load your javascript code, you can include your scripts by using <script src="YOUR_SCRIPT_PATH"></script> in the header section in your index.html. As jQuery and jQueryMobile are already loaded, you can any jQuery and jQueryMobile APIs in your script.

One thing you have to consider is that, your script can be run before all DOM nodes are loaded, which means that your DOM manipulation code cannot work. Therefore if you want to make sure that your script should be run after all DOM nodes are loaded, use domReady() function which is an opensource and included in tizen-web-ui-fw.js, like below;


	domReady(function() {
		// Put your code in here... 
	});

Now your code will be run after DOM tree is ready.
