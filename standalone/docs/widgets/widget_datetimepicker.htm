Date Picker, Time Picker, and Date Time Picker

The picker widgets show a control that you can use to enter date and time values.

Note
To auto-initialize form elements (without using picker widgets), use input elements with the type=date/time/datetime option within the form element.

To add a date time picker widget to the application, use the following code:

<li data-role="list-divider" data-line="false" class="ui-datetime-text-sub" > Date/Time Picker(Custom) - <span id="selected-date1"><em>(select a date first)</em></span></li>
<li class="ui-li-dialogue ui-datetime">
   <div class="ui-datetime-text-main">
      <input type="datetime" name="demo-date" id="demo-date" 
             data-format="MMM dd yyyy	hh:mm tt" value="2012-06-30T00:00:00+09:00" />
   </div>
</li>

The following table describes the picker options.

Option Input type Description
data-format String Sets the date and time format.

The default value is the system locale's date and time setting.
type String Defines the picker style. The style options are date (displays a date only), time (displays a time only), and datetime (displays both date and time).

The default value is datetime.
value String Sets the picker value.

The default value is the current system time.

The pickers can define a callback for the change event, which is fired when the picker value is changed:

<li data-role="list-divider" data-line="false"  class="ui-datetime-text-sub" > Data Format(MMMM) - <span id="selected-date6"><em>(select a date first)</em></span> </li>
<li class="ui-li-dialogue ui-datetime ui-datetime">
   <div class="ui-datetime-text-main">
      <input data-format="MMMM/dd/yyyy hh:mm tt" id="datetest" value="2013/1/31 15:08:44" type="datetime">
   </div>
</li>

<script>
   $("#datetest").bind( "change", function( )
   {
      $("#selected-date6").text($("#datetest").datetimepicker("value")) ;
   });
</script>

You can use the value method with the pickers to set or get the current date and time in the W3C DTF format:

$("#selected-date1").datetimepicker("value"); // 2012-05-30T01:23:45
$("#selected-date1").datetimepicker("value", "Jun 30, 2012 09:00");
