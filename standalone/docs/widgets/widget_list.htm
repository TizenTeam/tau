List

The list widget is used to display, for example, navigation data, results, and data entries. Lists can be used in Tizen as described in the jQueryMobile documentation for lists.

Note
A button widget (data-role="button") placed in the <a> tag is not supported in the list widget. The button must be placed in a <div> tag.

To add a list widget to the application, use the following code:

  • List with anchor

    You can add a list widget to the application with the anchor and press effect using the following code:

    <ul data-role="listview">
   <li> 
      <a href="#">1line</a>
   </li>
</ul>
  • List without anchor

    You can add a list widget to the application without the anchor and press effect using the following code:

    <ul data-role="listview">
   <li>1line</li>
</ul>
