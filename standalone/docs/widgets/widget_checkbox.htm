Checkbox

The checkbox widget shows a list of options on the screen where one or more can be selected. Checkboxes can be used in Tizen as described in the jQueryMobile documentation for checkboxes.

To add a checkbox widget to the application, use the following code:

<input type="checkbox" name="mycheck" id="check-test" class="favorite" />
<label for="check-test">Favorite</label>
<input type="checkbox" name="check-favorite" id="check-test2" 
 checked="checked" disabled="disabled" class="favorite" />
<label for="check-test2">Favorite Checked, Disabled</label>

The following table describes the checkbox options.

Option Input type Description
class String Defines the checkbox style.

The default value is check. If the value is set to favorite, a star-shaped checkbox is created.

The checkbox can define callbacks for events as described in the jQueryMobile documentation for checkbox events.

You can use methods with the checkbox as described in the jQueryMobile documentation for checkbox methods.
