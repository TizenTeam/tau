Flip Toggle Switch

The flip toggle switch widget shows a 2-state switch on the screen.

To add a flip toggle switch widget to the application, use the following code:

<select data-role="slider">
   <option value="off"></option>
   <option value="on"></option>
</select>

You can use methods with the flip toggle switch widget as described in the jQueryMobile documentation for flip toggle switch methods.
