Widget Options

The following table lists the additional jQueryMobile options that are supported for the widgets.

Widget Attribute Value
Button data-corners true or false
data-icon home, delete, plus, arrow-u, arrow-d, check, gear, grid, star, custom, arrow-r, arrow-l, minus, refresh, forward, back, alert, info, or search
data-iconpos left, right, top, bottom, or notext
data-inline true or false
Check box data-role None
Enhancement data-enhance true or false
data-ajax true or false
Fixed toolbar data-disable-page-zoom true or false
data-fullscreen true or false
data-visible-on-page-show true or false
Flip toggle switch data-role None
Footer data-id String: unique ID required for persistent footers.
data-position fixed
data-full-screen true or false
Header data-id String: unique ID required for persistent headers.
data-position fixed
data-full-screen true or false
Link data-ajax true or false
data-direction reverse: reverse transition animation (only for page or dialog).
data-dom-cache true or false
data-prefetch true or false
data-rel back: to move one step back in history.
data-transition fade, flip, flow, pop, slide, slidedown, slidefade, slideup, turn, or none
data-position-to origin, window, or jQueryObject
List view data-autodividers true or false
List view item data-role list-divider
Page data-add-back-btn true or false
data-back-btn-text String
data-dom-cache true or false
data-title String: title used when a page is shown.
data-url URL: value for updating the URL, instead of the URL used to request the page.
Popup data-tolerance 30, 15, 30, 15: distance from the edges of the window (top, right, bottom, left)
data-position-to origin, window, or jQueryObject
data-rel popup: for opening a pop-up
data-transition fade, flip, flow, pop, slide, slidedown, slidefade, slideup, turn, or none
Select data-icon home, delete, plus, arrow-u, arrow-d, check, gear, grid, star, custom, arrow-r, arrow-l, minus, refresh, forward, back, alert, info, search, or false
data-inline true or false
data-role None: prevents auto-enhancement from using the native control.
Slider data-highlight true or false: adds an active state fill on the track to handle.
data-role None: prevents auto-enhancement from using the native control.
Text input and text area data-role None: prevents auto-enhancement from using the native control.
