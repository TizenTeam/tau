Tabbar

The tabbar widget shows an unordered list of buttons in a header or footer wrapped together in a single group.

To add a tabbar widget to the application, use the following code:

<div data-role="tabbar">
   <ul>
      <li><a data-icon="edit">Tabbar1</a></li>
      <li><a data-icon="cancel">Tabbar2</a></li>
      <li><a data-icon="call">Tabbar3</a></li>
   </ul>
</div>

The following table describes the tabbar options.

Option Input type Description
data-icon <ICON_NAME> Sets an icon for each tabbar button.

The following list describes methods that can be used with tabbars:

  • enable

    The enable method is used to enable buttons in the tabbar list:

    • <div id="foo" data-role="tabbar"></div>

$("#foo").tabbar('enable', list_index);

  • disable

    The disable method is used to disable buttons in the tabbar list:

    • <div id="foo" data-role="tabbar"></div>

$("#foo").tabbar('disable', list_index);
