Gallery

The gallery widget shows images in a gallery on the screen.

To add a gallery widget to the application, use the following code:

<div data-role="gallery" id="gallery" data-vertical-align="middle" data-index="3">
   <img src="01.jpg">
   <img src="02.jpg">
   <img src="03.jpg">
   <img src="04.jpg">
   <img src="05.jpg">
   <img src="06.jpg">
   <img src="07.jpg">
   <img src="08.jpg">
</div>

The following table describes the image slider options.

Option Input type Description
data-index Number Defines the index number of the first image in the gallery.

The default value is 0.
data-vertical-align String Defines the image alignment. The alignment options are top, middle, and bottom.

The default value is top.

The following methods can be used with the gallery:

  • add

    The add method is used to add an image to the gallery. The image_file attribute defines the image file URL.

    • <div id="gallery" data-role="gallery" data-vertical-align="middle"></div>

$("#gallery").gallery('add', [image_file]);

  • remove

    The remove method is used to delete an image from the gallery. The image_index attribute defines the index of the image to be deleted.

    • <div id="gallery" data-role="gallery" data-vertical-align="middle"></div>

$("#gallery").gallery('remove', [image_index]);

  • refresh

    The refresh method is used to refresh the gallery. This method must be called after adding images to the image slider.

    • <div id="gallery" data-role="gallery" data-vertical-align="middle"></div>

$("#gallery").gallery('refresh');

  • empty

    The empty method is used to remove all the images from the gallery.

    • <div id="gallery" data-role="gallery" data-vertical-align="middle"></div>

$("#gallery").gallery('empty');

  • length

    The length method is used to get the length of all images in the gallery.

    • <div id="gallery" data-role="gallery" data-vertical-align="middle"></div>

$("#gallery").gallery('length');

  • value

    The value method is used to get or set the current index of the gallery.

    • <div id="gallery" data-role="gallery" data-vertical-align="middle"></div>

$("#gallery").gallery('value' [image_index]);
