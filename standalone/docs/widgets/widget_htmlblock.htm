HTML Block

The HTML block widget treats custom HTML as a widget. The user can insert any HTML command using this widget.

To add a HTML block widget to the application, use the following code:

<div id="htmlblock1">
   <!--HtmlCodeBlock-Start[#htmlblock1-->
      User Codes...
   <!--HtmlCodeBlock-End[#htmlblock1-->
</div>

The HTML block widget does not belong to the Tizen Web UI framework. The widget is supported by Tizen UI Builder that provides a customized HTML editor to edit the widget. You can open the editor using the HTML block's Properties view.
