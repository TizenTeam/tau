Search Filter Bar

The search filter bar widget is used to search for page content. This widget can be placed in the header or page content.

To add a search filter bar widget to the application, use the following code:

<label for="search-basic">Search Input:</label>
<input type="search" name="search" id="searc-basic" value="" data-mini="true" />

Tizen supports many search bar options as described in the jQueryMobile documentation for search bar options.

The search filter bar can define callbacks for events as described in the jQueryMobile documentation for search bar events.

You can use methods with the search filter bar as described in the jQueryMobile documentation for search bar methods.
