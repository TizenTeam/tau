List Divider

The list divider widget is used as a list separator for grouping lists. List dividers can be used in Tizen as described in the jQueryMobile documentation for list dividers.

Note
The list divider widget is supported since Tizen 2.0.

To add a list divider widget to the application, use the following code:

<li data-role="list-divider" data-style="check">
<form><input type="checkbox" name="c2line-check1" /></form></li>

The following table describes the list divider options.

Option Input type Description
data-style String Sets the style of the list divider. The style options are dialogue, check, and checkexpandable.

The list divider can define callbacks for events as described in the jQueryMobile documentation for list events.

You can use methods with the list divider as described in the jQueryMobile documentation for list methods.
